# 编译与执行

## 编译

cd tinykv make 或者 make kv

cd tinysql make 或者 make server

以上两个步骤，编译产生的可执行文件都在对应文件夹的 bin 目录下

## 启动

```

mkdir -p data


./tinyscheduler-server


./tinykv-server -path=data


./tinysql-server --store=tikv --path="127.0.0.1:2379"

```
